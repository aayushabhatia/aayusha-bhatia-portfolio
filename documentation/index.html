Setup Development Environment

To kickstart the development of your web app with this template, the first thing is to set up a React development environment.

Required tools:

  • Node JS

    • Download the latest Node.js version from nodejs.org

Introduction

To install this template locally, download and unzip the files. Open the 'react' folder with a code editor like VSCode.

Using npm:

In the terminal, run: npm install
Then: npm start
Access via localhost:3000.

Using yarn:

In the terminal, run: yarn install
Then: yarn start
Access via localhost:3000.

Folder Structure

  • public - Static files
    • images - All template images
    • index.html - Static entry point
  • src - React files
    • components - Reusable components
    • pages - Main pages
    • data - Template data
    • scss - Styling files
    • App.js - Main component
    • index.js - Entry point
